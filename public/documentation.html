This API builds a server-side component of the myFlix API

Request URL HTTP METHOD Request body data format Response body data format
Return a list of ALL movies to the user /movies GET

None

A JSON object holding data about all the movies
Return data (description, genre, director, image URL, whether it’s featured or not) about a single movie by title to the user /movies/[title] GET

None

A JSON object holding data about a single movie, containing a description, genre, director, image url, and whether it's featured or not. Example:
{_id: ObjectId("123"), Title: "Wonder Woman", Description: "...", Genre: {Name:"Action", Description: "..."}, Director: {Name:"Patty Jenkins",Bio:"...",Birth:"..."}, ImagePath: wonderwoman.png, Featured: True}

Return data about a genre (description) by name/title (e.g., “Thriller”) /movies/genre/[name] GET

None

A JSON object holding data about a genre, containing a description. Example:

{Genre: { Name: "Action", Description: "Action film is a film genre in which the protagonist or protagonists are thrust into a series of events that typically include violence, extended fighting, physical feats, rescues and frantic chases.", }}
Return data about a director (bio, birth year, death year) by name. /movies/directors/[name] GET

None

A JSON object holding data about a director, containing a bio, birth year, and death year. Example:

{Director: {Name:"Patty Jenkins",Bio:"...",Birth:"..."}, ImagePath: wonderwoman.png, Featured: True}
Allow new users to register /users POST

A JSON object holding data about the user to add, structured like:
{
username: "Anonymous"
favoriteMovies: { title:Batman, title: Spiderman 2 }}

A JSON object holding data about the new user that was added including a username.

Allow users to update their user info (username) /users/[username]/about PUT

None

A text message indicating the user info was updated
Allow users to add a movie to their list of favorites (showing only a text that a movie has been added—more on this later) /users/[username]/favorites PUT

A JSON object holding data about the movie to add to favorites, structured like: {favoriteMovie: {title: "Batman"}}

A JSON object holding data about the movie that was added to favorites.

Allow users to remove a movie from their list of favorites (showing only a text that a movie has been removed—more on this later) /users/[username]/favorites DELETE

None

A text message indicating whether the movie was successfully removed from their favorites.

Allow existing users to deregister (showing only a text that a user email has been removed—more on this later) /users/[username] DELETE

None

A text message indicating whether or not the username was deregistered.
